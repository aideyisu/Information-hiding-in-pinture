# Information-hiding-in-pinture
## 操作流程

登陆-上传源图片-后段处理生成图片(提供下载)

上传加密图片，生成解密文字.给予历史上传列表



## 计划

#### 程序写差不多（12-13号）：

上传加密+图像下载+图像上传解密+用户操作历史

13号：多种加密解密手法添加

#### 论文收工（14-15号 20页 + 2w字）

14号 多框架检测

存储设计+论文画流程图+基础框架定下来

#### 程序优化多解密方案提供（16-17号）

看看哪里可以优化

## 数据格式

历史行为表

| 字段名   | 解释                       |
| -------- | -------------------------- |
| uid      | 用户id                     |
| otype    | 操作类型(加密.解密1.解密2) |
| filename | 相关文件名                 |
| ctime    | 操作时间                   |
